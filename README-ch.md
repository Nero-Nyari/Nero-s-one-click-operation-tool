# 一键运行工具集

本工具集包含四个独立的小工具，涵盖图标获取、视频封面嵌入、音乐封面嵌入、音频格式转换等功能。

---

## 1. 获取图标工具  Icon Extractor

**文件位置：** ` Icon Extractor/获取图标工具.exe`

**功能说明：**
从可执行文件（.exe）、动态链接库（.dll）等文件中提取图标。

**使用方法：**

1. 双击运行 `获取图标工具.exe`
2. 选择需要提取图标的源文件
3. 选择图标保存位置
4. 程序将自动提取并保存图标文件

---

## 2. 视频文件读帧添加封面工具  Video Cover Embedder

**文件位置：** `Video Cover Embedder/视频封面嵌入工具.exe`

**功能说明：**
从视频文件中读取指定帧画面，并将该画面设置为视频的封面（缩略图）。适用于为视频文件添加自定义封面图。

**使用方法：**
1. 双击运行 `视频封面嵌入工具.exe`
2. 选择需要添加封面的视频文件
3. 选择或指定要使用的封面图片
4. 程序将自动将封面嵌入到视频文件中

---

## 3. 音乐文件添加封面工具  Music Cover Embedder

**文件位置：** `Music Cover Embedder/MusicCoverEmbedder.exe`

**功能说明：**
为音乐文件（如 MP3、FLAC 等）嵌入封面图片。嵌入后在播放器中即可显示专辑封面。

**使用方法：**
1. 双击运行 `MusicCoverEmbedder.exe`
2. 选择需要添加封面的音乐文件
3. 选择要嵌入的封面图片
4. 程序将自动将封面信息写入音乐文件

---

## 4. 音频 MP4 转 MP3 工具  MP4 to MP3 Converter

**文件位置：** `MP4 to MP3 Converter/mp4_to_mp3.exe`

**功能说明：**
将 MP4 格式的音频文件转换为 MP3 格式，方便在更多设备和播放器上使用。

**使用方法：**
1. 双击运行 `mp4_to_mp3.exe`
2. 选择需要转换的 MP4 文件
3. 选择输出文件的保存位置
4. 程序将自动完成格式转换

---

## 注意事项

- 所有工具均为独立运行，无需安装，双击即可使用。
- 建议在转换或嵌入操作前，先备份原始文件，以防操作不可逆。
- 如遇到杀毒软件误报，请将工具目录添加到信任列表。
