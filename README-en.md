# One-Click Toolset

A collection of four standalone utility tools covering icon extraction, video cover embedding, music cover embedding, and audio format conversion.

---

## 1. Icon Extractor

**File Location:** ` Icon Extractor/获取图标工具.exe`

**Description:**
Extracts icons from executable files (.exe), dynamic-link libraries (.dll), and other similar file types.

**Usage:**

1. Double-click `获取图标工具.exe` to launch
2. Select the source file from which you want to extract the icon
3. Choose a save location for the extracted icon
4. The program will automatically extract and save the icon file

---

## 2. Video Cover Embedder

**File Location:** `Video Cover Embedder/视频封面嵌入工具.exe`

**Description:**
Reads a specific frame from a video file and sets it as the video's cover/thumbnail. Ideal for adding custom cover images to video files.

**Usage:**
1. Double-click `视频封面嵌入工具.exe` to launch
2. Select the video file you want to add a cover to
3. Choose the cover image to use
4. The program will automatically embed the cover into the video file

---

## 3. Music Cover Embedder

**File Location:** `Music Cover Embedder/MusicCoverEmbedder.exe`

**Description:**
Embeds cover images into music files (MP3, FLAC, etc.). Once embedded, album artwork will be displayed in media players.

**Usage:**

1. Double-click `MusicCoverEmbedder.exe` to launch
2. Select the music file you want to add a cover to
3. Choose the image to embed
4. The program will automatically write the cover information into the music file

---

## 4. MP4 to MP3 Converter

**File Location:** `MP4 to MP3 Converter/mp4_to_mp3.exe`

**Description:**
Converts MP4 audio files to MP3 format for broader compatibility across devices and media players.

**Usage:**

1. Double-click `mp4_to_mp3.exe` to launch
2. Select the MP4 file to convert
3. Choose the output save location
4. The program will automatically complete the format conversion

---

## Notes

- All tools are standalone and portable — no installation required, just double-click to run.
- It is recommended to back up your original files before performing any conversion or embedding operations, as these actions may be irreversible.
- If your antivirus software flags any of the tools, please add the tool directory to the trusted/exception list.
